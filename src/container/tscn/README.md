# tscn

github: [https://github.com/marioa5945/tscn](https://github.com/marioa5945/tscn)

## Installation

```sh
# Locally in your project
yarn add tscn --dev

npm install -D tscn

# Or globally
yarn global add tscn

npm install -g tscn
```

## Usage

### Shell

```sh
# Execute a script as `node` + `tsc`.
tscn script.ts [-w]
```

### Debug

```sh
npm run dev
```

### watch

watch files: ./server/?(.ts | .js | .mjs)
restart delay: 1500ms
